setTimeout, setInterval and clearTimeout

  1. Delays an action
  2. Executes an action in a regular basis
  3. Stops an action started with setInterval

open

Opens a new url

resizeBy and resizeTo

  1. Adds or subtracts from the window size
  2. Stablishes the new size

moveTo

move an already opened window

close

Close an already opened window

showing information

alert, confirm and prompt

  1. alert: Shows a message in a modal window
  2. confirm: Shows a confirmation window
  3. prompt: Asks for data
