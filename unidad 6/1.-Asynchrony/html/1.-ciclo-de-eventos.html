Ejemplos de orden de ejecución en el ciclo de eventos

Ejemplo 1: Código síncrono y código asíncrono (promesas)


            console.log('Código síncrono 1');

            Promise.resolve().then(() => {
                console.log('Promesa resuelta');
            });

            console.log('Código síncrono 2');

Ejemplo 2: Código síncrono, asíncrono urgente (Promesas) y asíncrono no urgente (setTimeout)


            console.log('Código síncrono 1');

            setTimeout(() => {
                console.log('setTimeout');
            }, 0);

            Promise.resolve().then(() => {
                console.log('Promesa resuelta');
            });

            console.log('Código síncrono 2');
